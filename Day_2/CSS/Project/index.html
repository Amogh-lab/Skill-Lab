Discrete Maths

Topics include logic, set theory, graph theory, combinatorics, and number theory essential for computer science.

AIML

Focuses on Artificial Intelligence and Machine Learning algorithms, models, and real-world applications.

Computer Network

Covers network architecture, protocols, TCP/IP, OSI model, and basic routing principles.

DAA

Design and Analysis of Algorithms helps build efficient and optimized solutions using divide & conquer, greedy, and dynamic programming techniques.

DSDA

Data Science and Data Analysis explores preprocessing, EDA, visualization, and data-driven decision-making processes.

Cyber Physical System

Focuses on systems where physical and software components are deeply intertwined such as IoT, embedded systems, and real-time applications.
